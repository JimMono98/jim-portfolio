## About

Personal portfolio website built with modern web technologies, showcasing my journey as a full-stack developer with over 8 years of experience building web applications, APIs, and digital solutions.

---

## Features

- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Smooth Animations** - Page transitions powered by Framer Motion
- **Typing Effect** - Dynamic homepage introduction with Typed.js
- **Project Showcase** - Interactive Swiper slider gallery
- **Contact Form** - Functional email integration via Nodemailer
- **Skills Grid** - Interactive tooltips displaying technical expertise
- **Resume Tabs** - Experience, education, and skills organized in tabs

## Getting Started

```bash
# Clone the repository
git clone https://github.com/JimMono98/jim-portfolio.git

# Navigate to directory
cd jim-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
